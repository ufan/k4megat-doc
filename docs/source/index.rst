Welcome to k4megat documentation!
===================================

**k4megat** is the framework software for simulation, recostruction and analysis of
**Megat** project.
**Megat** is a space gamma spectrometer focusing on sub-MeV energy range.

.. note::

   This project is still in its early stage, designning and prototyping.
   Everything is under active development.
   If wanna more details of the underlying **Key4hep** packags, `read Dive-series <https://k4megat-doc.readthedocs.io/projects/third-party/en/latest>`_
